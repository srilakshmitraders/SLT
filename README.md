# Sri Lakshmi Traders Website

Premium responsive static website for **Sri Lakshmi Traders**, a wholesale cement dealer based in Visakhapatnam, India.

## Project Structure

```text
SLT/
├── index.html
├── README.md
├── assets/
│   ├── favicon.svg
│   └── logo.svg
├── css/
│   └── styles.css
└── js/
    └── main.js
```

## Features

- Pure HTML, CSS, and JavaScript
- Responsive desktop and mobile layout
- Premium dark, teal, and gold visual theme
- Custom stylized `SL` logo and favicon inspired by the visiting card
- Smooth scrolling, reveal animations, hover transitions
- About, services, founder, awards, products, premises, contact, and location sections
- Google Maps link and embedded map preview
- SEO and social preview meta tags

## Run Locally

Open `index.html` directly in a browser. No build step is required.

## Deploy to GitHub Pages

1. Create a new public GitHub repository.
2. Upload all files from this folder to the repository.
3. Go to **Settings** → **Pages**.
4. Under **Build and deployment**, select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save and open the published GitHub Pages URL after deployment finishes.
